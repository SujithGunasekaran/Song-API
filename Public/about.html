About Song API
GET :
  • URL : http://localhost:5000/songList/get/songs ( development Purpose ).
  • If Your going to make public api http:'Your url'/songList/get/songs
  • If there is 40,000 data in your api result, page gets slow to load. so for that I have made query params in request
    URL : http://localhost:5000/songList/get/songs?maxResult=10
    If we want only 10 result initially we can send maxResult=10, If you have not send maxResult query param by default it will send 1000 result initially.
  • If There is a need to filter data, we have an option for that.
    URL : https://localhost:5000/songList/get/songs?filmName="Thani%20Oruvan"
    if you send query param of filmName in request. You will be getting only the filmName data which you have given
